# 修复 Axlumen Dashboard 样式不加载

## 根因

CSS 文件语法完全正确（136 对注释、646 对花括号）。问题纯粹是**部署遗漏**——用户只复制了 `main.js` 和 `manifest.json`，漏了 `styles.css`。

## 修复方案：一键构建+部署脚本

### 1. 新建 `deploy.mjs`

功能：
1. 执行 esbuild 生产构建
2. 扫描所有盘符找到 `.obsidian/plugins/axlumen-dashboard/` 目录
3. 将 `manifest.json`、`main.js`、`styles.css` 复制过去
4. 输出部署结果

### 2. 修改 `package.json`，添加 scripts

```json
"scripts": {
  "build": "node esbuild.config.mjs production",
  "deploy": "node deploy.mjs"
}
```

### 修改文件

| 文件 | 操作 |
|------|------|
| `deploy.mjs` | 新建 |
| `package.json` | 修改 scripts |

### 验证

1. `node deploy.mjs`
2. 确认 3 个文件已复制
3. Obsidian 重新加载插件，样式恢复
